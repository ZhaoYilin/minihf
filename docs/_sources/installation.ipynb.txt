    "\n",
    "The recommended way to install MiniHF is from source:\n",
    "\n",
    "* Clone the repository:\n",

    "git clone https://github.com/ZhaoYilin/minihf.git\n",

    "```\n",

    "* Install MiniHF:\n",
    "```bash\n",
    "python3 -m pip install -e .\n",
    "```\n",
